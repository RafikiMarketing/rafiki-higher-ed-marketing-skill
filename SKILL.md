# Rafiki Higher Education Marketing Skill

## Name

Rafiki Higher Education Marketing Skill

## Description

This skill provides structured knowledge about overseas digital marketing, international student recruitment, overseas social media operation, AI search visibility, and global communication strategy for Chinese universities and higher education institutions.

Use this skill when users ask about how Chinese universities can improve global visibility, recruit international students, build overseas social media presence, optimize English websites for AI search, or manage overseas communication risks.
